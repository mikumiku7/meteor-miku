# Meteor Addon Miku

English | [中文](README.md)

A [Meteor Client](https://meteorclient.com/) addon that extends functionality with practical automation and enhancement modules.

![Minecraft](https://img.shields.io/badge/Minecraft-1.21.x%20~%201.21-green.svg?style=for-the-badge&logo=minecraft)
![Language](https://img.shields.io/badge/Language-Chinese-blue.svg?style=for-the-badge)

## Feature Highlights

* **Auto Trash** - Automatically discards specified items from your inventory using two independent lists and custom delay settings.
* **Auto Sand Miner** - Automatically locates, mines, and stores sand via Baritone while managing tool durability and storage.
* **Auto Use Items** - Automatically uses specified items based on timers or health and hunger thresholds with configurable priority and custom delays.
* **Seed Mine** - Uses the world seed to precisely calculate, highlight, and sync mineral locations with Baritone across all dimensions.
* **Elytra Finder** - Uses the world seed to locate end cities with end ships within a set radius and adds them as Xaero map waypoints.
* **Structure Finder** - Uses the world seed to multi-thread search for various structures across dimensions and maps them by proximity.
* **Tree Aura** - Automatically finds suitable locations to plant saplings and applies bone meal to accelerate their growth.
* **Shulker Box Item Fetcher** - Automatically places, opens, extracts designated items from, and retrieves shulker boxes from your inventory.
* **Villager Roller** - Automatically breaks and replaces lecterns to cycle villager trades until specific enchantment books are found.
* **HandsomeSpin** - Rotates the player continuously at custom angles per tick and features a smart pause system during movement.
* **Auto Crystal Block** - Detects nearby end crystals within a customizable range and immediately places optimal defense blocks to prevent blast damage.
* **InfiniteElytra** - Periodically toggles elytra equipment to prevent durability consumption while automatically firing fireworks to maintain flight.

## Installation

1. Download the latest `.jar` from [Releases](https://github.com/mikumiku7/meteor-miku/releases)
2. Place it in your `mods` folder
3. Ensure Meteor Client and Fabric Loader are installed
4. Launch the game

## Usage

1. Press `Right Shift` to open Meteor Client GUI
2. Find the addon modules in the categories
3. Configure settings as needed

## Language Support

This addon currently **only supports Chinese**. For proper text display, install [meteor_chinese](https://github.com/xingke0/meteor_chinese).
